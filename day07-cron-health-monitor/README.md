# Automated Server Health Monitoring

This project uses Bash scripting and Cron Jobs to automate Linux server health monitoring.

## Features

- CPU Monitoring
- Memory Monitoring
- Disk Monitoring
- Process Monitoring
- Automated Reports
- Logging
- Cron Automation

## Run

chmod +x scripts/health_report.sh

./scripts/health_report.sh

## Setup Cron

chmod +x setup.sh

./setup.sh
